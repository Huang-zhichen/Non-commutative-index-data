# Non-commutative-index-data
This repository contains the numerical data supporting the article "Noncommutative index of measurement-only entanglement phase transitions"by Zhichen Huang, Chunxiao Du, Yang Zhou, Zhisong Xiao.
Here, we provide raw data of our simulation. For every models with different r, the parameters' setting, von-Neumann entropy in every 500 time steps, steady state von-Neumann entropy both half cutting and a sepecific subregion, mutual information, tripartite information and support distribution of stabilizer in clipped gauge are provided
